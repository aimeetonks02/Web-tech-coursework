Summer Work

Scrimba Course

Over the summer of 2022, I decided to undergo a course on HTML and CSS from Scrimba.com in order to better prepare myself for the Web Technologies module that I knew I had in the first semester of my second year.

This course improved upon the basics that I had learnt during my Introduction to Interaction Design module in my first year. I chose this course in particular so that I could better learn how to make my own websites as it was one of my favourite things I learnt in the past year.

This course has therefore helped me to get a good start on my Web Technologies coursework. This was beneficial as I had a better understanding coming into the module and could wrap my head around the new content easier.

GitHub

Over the summer, I also worked to set up a GitHub for myself so that I can keep a professional portfolio of all of my university programming. I also hope that by learning GitHub now, I can use it as a skill to stand out from other candidates when applying for future jobs or graduate academies.

GitHub also allows me to keep track of any time I update the code that I am working on, which is beneficial as I am able to go back to previous versions of my code. If something were to break, this would allow me to figure out what I changed that caused the error to occur.
